Tecnologias Utilizadas

  • Django: Framework web em Python usado para desenvolver o site.
  • HTML e CSS: Linguagens básicas para estrutura e estilo do site.
  • JavaScript: Usado para interatividade, como o botão de alternar o modo escuro.
  • Python: Linguagem de programação utilizada com Django para criar a lógica de backend do site.
  • Git: Sistema de controle de versão para colaboração e controle de código-fonte.
  • GitHub: Plataforma de hospedagem de código-fonte para o projeto.

Estrutura do Projeto

O projeto segue a estrutura padrão do Django, com os seguintes diretórios e componentes principais:

  • portfolio: Aplicação principal que contém as configurações do Django e as views.
  • models: Definição das estruturas de dados e a interação com o banco de dados.
  • views: Funções que gerenciam a lógica do aplicativo e retornam as respostas HTML.
  • urls: Sistema de roteamento que direciona as URLs para as views correspondentes.
  • loader: Funções ou classes usadas para carregar dados iniciais ou arquivos específicos.
  • json: Utilizado para trabalhar com dados no formato JSON, especialmente em APIs e troca de dados assíncronos.
  • static: Diretório para arquivos estáticos, como CSS, JS e imagens.
  • templates: Diretório para os templates HTML usados no site.
